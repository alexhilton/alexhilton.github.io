数列

数列问题也比较常见，虽然它们一般并不是直接出现的，但很多问题可以转化为数列问题。

等差数列

首项a1, 未项an，公差d=a2 - a1，an=a1+(n-1)d，前n项的和为：Sn=n(a1+an)/2。 自然数，首项是1，未项是n，所以Sn=(n+1)n/2。

等比数列

Sn = a1(1-qn)/(1-q)，如果公比是1，则Sn=n x a1。
